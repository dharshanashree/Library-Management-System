# Library Management System

A desktop-based Library Management System built with Java Swing, applying Object-Oriented Programming principles for efficient library operations.

## Features
- Add, update, and delete book records
- Search books by title or author
- Issue and return book tracking
- Clean GUI built with Java Swing

## Tech Stack
- Java 21
- Java Swing (GUI)
- OOP Concepts: Encapsulation, Inheritance, Abstraction, Polymorphism

## How to Run
```bash
javac Main.java
java Main
```
